# COMP30880_ComicBoys


## How to update dependencies in Maven project
#### Go to project directory (same directory as pom.xml)
```bash
mvn clean install
```

## How to run Maven project
#### Go to project directory (same directory as pom.xml)
```bash
mvn clean package
java -jar target/COMP30880_ComicBoys-1.0-SNAPSHOT.jar
```
